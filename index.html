API Docs

GET /api/search

Cara pakai API: 
    GET https://melolo-api.vercel.app/api/search?query=super+keren&offset=0&limit=10
Response: 
{
    "status_code": 200,
    "books": [
        {
        "series_id": "12345",
        "title": "Judul Novel",
        "last_chapter_index": "10",
        "thumb_url": "https://..."
        }
    ]
    }
Example Requests: 
import requests
    res = requests.get("https://melolo-api.vercel.app/api/search", params={"query":"super keren","offset":0,"limit":10})
    print(res.json())
fetch("https://melolo-api.vercel.app/api/search?query=super+keren&offset=0&limit=10")
    .then(res=>res.json())
    .then(data=>console.log(data))
<?php
    $res = file_get_contents("https://melolo-api.vercel.app/api/search?query=super+keren&offset=0&limit=10");
    echo $res;
curl "https://melolo-api.vercel.app/api/search?query=super+keren&offset=0&limit=10"
